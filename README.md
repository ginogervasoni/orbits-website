# ORBITS

Landing page oficial de ORBITS Academy, preparada para publicarse como sitio estático en Vercel.

## Publicación

- `index.html`: sitio completo.
- `vercel.json`: reglas de reescritura para que `/`, `/01` ... `/100` carguen la misma landing.

## Códigos de origen

- `/` genera códigos terminados en `-000`.
- `/01` genera códigos terminados en `-01`.
- `/100` genera códigos terminados en `-100`.

El código se genera en el navegador y sirve para atribución operativa. No reemplaza una base de datos ni evita modificaciones manuales del código por parte del usuario.
